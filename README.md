oxide colorscheme for [wezterm](https://wezterm.org/).

## Installation

1. Download `oxide.toml` from this repository
2. Copy it to your wezterm config directory: `~/.config/wezterm/colors/`
3. Set the scheme in your `wezterm.lua`:

```lua
return {
  color_scheme = "oxide",
}
```
